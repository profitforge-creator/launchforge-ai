# OAuth Redirects

Status date: 2026-06-15

## Current Flow

LaunchForge starts Google sign-in through Supabase Auth:

```ts
supabase.auth.signInWithOAuth({
  provider: "google",
  options: {
    redirectTo: "<app-origin>/api/auth/supabase/callback"
  }
})
```

Google does not redirect directly to the LaunchForge app. Google redirects to Supabase:

```text
https://kphhqxjfmkkutmtqnxkp.supabase.co/auth/v1/callback
```

After Supabase handles Google, Supabase redirects back to LaunchForge:

```text
<app-origin>/api/auth/supabase/callback
```

## Exact Local URLs

Local app callback generated by LaunchForge when `NEXT_PUBLIC_APP_URL` is unset:

```text
http://localhost:3000/api/auth/supabase/callback
```

Local password reset redirect:

```text
http://localhost:3000/login
```

## Exact Production URLs

Canonical production origin:

```text
https://launchforge-ai-six.vercel.app
```

Set this in Vercel Production:

```env
NEXT_PUBLIC_APP_URL=https://launchforge-ai-six.vercel.app
```

Then LaunchForge generates the Supabase callback:

```text
https://launchforge-ai-six.vercel.app/api/auth/supabase/callback
```

Password reset redirect:

```text
https://launchforge-ai-six.vercel.app/login
```

GitHub OAuth callback:

```text
https://launchforge-ai-six.vercel.app/api/auth/github/callback
```

Stripe Connect OAuth callback:

```text
https://launchforge-ai-six.vercel.app/api/auth/stripe/callback
```

Webflow OAuth callback:

```text
https://launchforge-ai-six.vercel.app/api/auth/webflow/callback
```

## Add To Google Cloud OAuth App

Authorized redirect URI:

```text
https://kphhqxjfmkkutmtqnxkp.supabase.co/auth/v1/callback
```

Do not add the LaunchForge app callback as the Google OAuth redirect URI. Supabase is the OAuth client that talks to Google.

## Add To Supabase Auth URL Allowlist

Local:

```text
http://localhost:3000/api/auth/supabase/callback
http://localhost:3000/login
http://localhost:3000/dashboard
```

Vercel preview:

```text
https://<your-vercel-preview-domain>/api/auth/supabase/callback
https://<your-vercel-preview-domain>/login
https://<your-vercel-preview-domain>/dashboard
```

Vercel production:

```text
https://launchforge-ai-six.vercel.app/api/auth/supabase/callback
https://launchforge-ai-six.vercel.app/login
https://launchforge-ai-six.vercel.app/dashboard
```

If using Vercel branch preview URLs, add each preview origin or an approved Supabase wildcard pattern before testing OAuth.
